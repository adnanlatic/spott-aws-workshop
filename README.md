## Docker
add this project in host file

here you can find your host file:

Windows: /Windows/system32/drivers/etc/hosts

Linux/OSX: /etc/hosts

```sh
127.0.0.1 spott-aws-workshop
```

checkout docker branch spott-aws-workshop

```sh
docker compose down
docker compose up -d
```
